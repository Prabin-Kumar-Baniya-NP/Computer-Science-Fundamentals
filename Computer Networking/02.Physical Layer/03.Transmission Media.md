# Transmission Media

A communication channel that carries the information from sender to receiver.

## Classification of Transmission Media

1. Guided Media
2. Unguided Media

### 1. Guided Media

1. Coaxial Cable
2. Fibre Optics Cable
3. Twisted Pair Cable

### 2. Unguided Media

1. Radiowaves
2. Microwaves
3. Infrared
